What is "Covid Stats EU"?

Covid Stats EU is a collaborative university assignment and it's made by Dimitris Pramateftakis and Alexandros Rigas.

This web application contains Covid-19 data up to March of 2021. It doesn't update it self :(.

The API is made with express.js and the frontend is made with pure HTML CSS JS. You can view the code of this project over on github.
